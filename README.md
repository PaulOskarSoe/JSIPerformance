# JSIPerformance

### Run project on Android

**Requirements**

- Android Studio
- Android Emulator or physical device
- Node from version 12
- npm or yarn package manager 

**Step one: install node_modules**

Install with yarn

```shell
yarn install
```

Install with npm
```shell
npm install
```

**Step two: build code**

Build with yarn

```shell
yarn run android
```

Build with npm

```shell
npm run android
```
